# Console-Based Todo List App (JavaScript)

This is a simple console-based Todo List application built using JavaScript. It allows users to manage their daily tasks directly from the console.

## Features

- Add new tasks
- View all tasks
- Delete tasks by selecting task number
- Quit the application safely
- Beginner-friendly and clean code

## Concepts Used

- Arrays
- Loops (while, for)
- Conditional statements
- User input using prompt()
- Array methods like push() and splice()

## How it works

The application continuously asks the user to choose an action:

- `add` → Adds a new task
- `list` → Shows all tasks
- `delete` → Deletes a selected task
- `quit` → Exits the application

## Example

Enter: add
Enter task: Study JavaScript

Enter: list

1. Study JavaScript

Enter: delete
Enter task number: 1

Task deleted successfully.
